## 🔍 三维看市场

> 判断大盘环境，筛选合适时机

| 维度 | 分析内容 | 数据来源 |
|------|---------|---------|
| **消息** | 政策、宏观、行业、公司公告 | mx-macro-data, mx-finance-search |

| 信号组合 | 市场判断 | 操作 |
|---------|---------|------|
| 消息🟢 + 情绪🟢 + 技术🟢 | 强势上涨 | 积极做多 |
| 消息🟢 + 情绪🟡 + 技术🟢 | 震荡偏强 | 精选个股 |
| 消息🔴 + 情绪🔴 + 技术🔴 | 弱势下跌 | 休息/空仓 |
| 消息🟡 + 情绪🟡 + 技术🟡 | 震荡整理 | 高抛低吸 |

## ⭐ 四维选股评级

> 评估个股机会，星级越高越好

| 维度 | 星级 | 说明 |
|------|------|------|
| **机会⭐** | 1-5 | 上涨空间与概率 |
| **安全⭐** | 1-5 | 风险程度（R3红线） |
| **确定⭐** | 1-5 | 上涨逻辑清晰度 |
| **舒适⭐** | 1-5 | 持有体验（位置/波动） |

### 评级决策树

```
安全⭐ < 3 → ⭐ 不买（R3红线触发）
         │
         ▼
安全⭐ ≥ 3 + 机会⭐ ≥ 3 + 确定⭐ ≥ 3 → ⭐⭐⭐⭐⭐ 可以买
```

### 综合评级计算

```
综合分 = 机会⭐×30% + 安全⭐×25% + 确定⭐×25% + 舒适⭐×20%

4.5+ → ⭐⭐⭐⭐⭐ 立即买
3.5+ → ⭐⭐⭐⭐ 可以买
2.5+ → ⭐⭐⭐ 观望
```

> 详细评分规则：[references/core/rating/](references/core/rating/)

## 📋 输出规范

### 必备内容

| # | 内容 | 示例 |
|---|------|------|
| 1 | 具体价位 | 买入价XX、止损价XX、目标价XX |
| 2 | 置信度 | 高/中/低 + 理由 |
| 3 | 核心逻辑 | 为什么现在买 |
| 4 | 时间窗口 | 验证周期（1-3日） |
| 5 | 仓位建议 | 首仓X成、最大X成 |

### 标准格式

```markdown
## [股票] 诊断报告

**评级**：⭐⭐⭐⭐ | **置信度**：中高
**三维**：消息🟢 情绪🟢 技术🟢
**四维**：机会4 安全4 确定5 舒适3

| 参数 | 价位 | 说明 |
|------|------|------|
| 买入价 | XX.XX | 突破确认后介入 |
| 止损价 | XX.XX | 跌破必须走 |
| 目标价 | XX.XX | 止盈参考 |
| 仓位 | X成 | 不超过总仓位X成 |

*本报告仅供参考，不构成投资建议*
```

## 🚫 硬规则

违反以下任一规则，直接取消资格：

| # | 规则 | 原因 |
|---|------|------|
| 1 | **R3红线** | 安全⭐<3 禁止推荐 |
| 2 | **止损7%** | 不执行止损是亏损主因 |
| 3 | **T+1** | 当日买不可卖 |
| 4 | **流动性** | 成交额<5000万不做主要标的 |
| 5 | **仓位** | 单只≤30%，总≤50% |
| 6 | **Disclaimer** | 必须注明仅供参考 |
