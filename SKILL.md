## 🔬 四维分析法 (4D Analysis)

### 维度A：消息/政策（News & Policy）

**数据源**：
- `mx-finance-search` — MX 宏观与行业政策
- **东方财富（浏览器）** — https://www.eastmoney.com/news/latest.html
- `akshare` — 财经新闻、公告

**分析重点**：
- 宏观政策：央行货币政策、证监会公告、监管动向
- 行业政策：发改委/工信部政策、顶层意见（中办/国办）
- 突发事件：地缘政治、疫情、灾难
- 公司公告：业绩预告、并购重组、股权激励

**判断标准**：
| 信号 | 含义 |
|------|------|
| 消息提前消化 | 可能见光死 |
| 超预期催化 | 可能继续涨 |
| 辟谣/澄清 | 可能是假消息 |

### 维度B：情绪/资金（Sentiment & Capital）

**数据源**：
- `taoguba-hot` — 淘股吧散户情绪
- `mx-data` — MX 资金流向
- **东方财富（浏览器）** — 涨跌停、炸板、连板数据
- **AkShare** — 北向资金、板块资金流

**情绪指标**：
| 指标 | 说明 | 信号 |
|------|------|------|
| 涨停家数 | 市场情绪温度计 | >50过热 / <20冰点 |
| 昨日涨停股今日表现 | 赚钱效应 | 平均溢价>5%强势 |
| 炸板率 | 多空分歧 | >30%情绪不稳 |
| 龙虎榜 | 游资动向 | 营业部大规模买入 |

**资金流向**：
| 信号 | 含义 |
|------|------|
| 主力净流入 + 散户净流出 | 洗盘嫌疑，可能是机会 |
| 主力净流出 + 散户净流入 | 主力出货，风险大 |
| 北向资金大幅流入 | 外资看多，跟随信号 |

### 维度C：流动性/技术（Liquidity & Technical）

**数据源**：
- `mx-finance-data` — MX 技术指标
- **AkShare** — MA、MACD、KDJ、RSI、布林带
- **东方财富（浏览器）** — 实时行情、成交明细、资金流入

**成交量模式**：
| 模式 | 阈值 | 策略含义 |
|------|------|---------|
| 天量 | >2.0万亿 | 警惕，可能顶部或突破 |
| 正常 | 1.0-2.0万亿 | 正常交易环境 |
| 缩量 | <0.8万亿 | 观望为主，找确定性机会 |

**技术位判断**：
| 价格位置 | 信号 |
|---------|------|
| 缩量回踩均线（MA5/MA10） | 较好买点 |
| 放量突破前高 | 关注跟进 |
| 高位放量大阴线 | 警惕见顶 |
| 一字板后炸板 | 大幅回落风险 |

### 维度D：风险/合规（Risk & Compliance）

**数据源**：
- `scripts/check_risk.py` — 强制风控扫描
- **东方财富（浏览器）** — 风险提示公告、立案调查、减持计划
- **AkShare** — ST股、退市风险、财务风险

**安全红线（绝对禁止）**：
| 风险类型 | 描述 | 后果 |
|---------|------|------|
| 立案调查 | 证监会立案 | 🚫 禁止推荐 |
| 退市风险 | *ST、**ST | 🚫 禁止推荐 |
| 大比例减持 | 股东减持>5% | 🚫 禁止推荐 |
| 业绩巨亏 | 亏损超预期 | 🚫 禁止推荐 |

**业绩窗口期**：
| 时间 | 风险 |
|------|------|
| 4月20-30日 | 年报+一季报窗口，高位股安全降为⭐ |
| 8月20-31日 | 中报窗口 |
| 10月20-31日 | 三季报窗口 |

## ⚖️ 评级体系（统一五星制）

> 所有维度均采用 1-5 星评级，**星越高越好**，与直觉一致

---

### ⭐ 安全评级（Safety）

**含义**：股票的风险程度 — 星越高越安全

| 星级 | 含义 | 描述 |
|:----:|------|------|
| ⭐ | 极危险 | R3红线触发，随时可能踩雷 |
| ⭐⭐ | 较危险 | 存在明显风险因素，需高度警惕 |
| ⭐⭐⭐ | 中等 | 存在一定风险，谨慎操作 |
| ⭐⭐⭐⭐ | 较安全 | 风险可控，可关注 |
| ⭐⭐⭐⭐⭐ | 极安全 | 风险极低，优质标的 |

---

### ⭐ 机会评级（Opportunity）

**含义**：股票的潜在机会 — 星越高机会越大

| 星级 | 含义 | 描述 |
|:----:|------|------|
| ⭐ | 机会极小 | 缺乏催化剂，行情难以启动 |
| ⭐⭐ | 机会较小 | 有一定机会但不确定 |
| ⭐⭐⭐ | 中等机会 | 有逻辑支撑的机会 |
| ⭐⭐⭐⭐ | 较大机会 | 有明确催化因素 |
| ⭐⭐⭐⭐⭐ | 极大机会 | 龙头潜力，板块共振 |

---

### ⭐ 确定评级（Certainty）

**含义**：分析和信号的确定程度 — 星越高越可靠

| 星级 | 含义 | 描述 |
|:----:|------|------|
| ⭐ | 很不确定 | 纯赌方向，信号模糊 |
| ⭐⭐ | 确定性较低 | 信号较弱，需观察 |
| ⭐⭐⭐ | 中等确定 | 有一定信号支撑 |
| ⭐⭐⭐⭐ | 较高确定 | 信号明确，可信度高 |
| ⭐⭐⭐⭐⭐ | 极高确定 | 信号强烈，多维度共振 |

---

### ⭐ 舒适评级（Comfort）

**含义**：买点的舒适程度 — 星越高买点越舒服

| 星级 | 含义 | 描述 |
|:----:|------|------|
| ⭐ | 很不舒适 | 一字板、高位炸板、业绩盲盒 |
| ⭐⭐ | 较不舒适 | 追涨介入，成本劣势 |
| ⭐⭐⭐ | 一般舒适 | 非最佳但可接受的位置 |
| ⭐⭐⭐⭐ | 较舒适 | 回踩均线、缩量整理后企稳 |
| ⭐⭐⭐⭐⭐ | 极佳舒适 | 底部分歧转一致、缩量回踩均线、龙头缩量封板 |

---

### 综合评级速查

| 组合 | 评价 | 操作建议 |
|------|------|---------|
| 安全⭐⭐⭐⭐⭐ + 机会⭐⭐⭐⭐⭐ + 确定⭐⭐⭐⭐ + 舒适⭐⭐⭐⭐⭐ | 完美标的 | **重仓参与** |
| 安全⭐⭐⭐⭐ + 机会⭐⭐⭐⭐ + 确定⭐⭐⭐ | 优质标的 | 正常参与 |
| 安全⭐⭐⭐ + 机会⭐⭐⭐ | 一般标的 | 小仓位试探 |
| 安全⭐⭐ + 机会⭐⭐ | 风险较大 | 观望或不参与 |
| 安全⭐ + 任何机会 | 高危区域 | **禁止推荐** |

## 🚫 主动避开的错误（Anti-Patterns）

### 致命错误（必死无疑）

| # | 错误 | 典型后果 |
|---|------|---------|
| 1 | **安全⭐股票仍推荐买入** | 被套、踩雷、退市 |
| 2 | **高位追一字板** | 次日低开-15%起步 |
| 3 | **逆势抄底** | 接飞刀、越套越深 |
| 4 | **T+1当天买当天卖** | 违规无法成交 |
| 5 | **忽视流动性** | 想卖卖不出，成交价极差 |

### 高危错误（大概率亏损）

| # | 错误 | 典型后果 |
|---|------|---------|
| 6 | **消息兑现追涨** | 利好出尽变利空 |
| 7 | **忽视板块联动** | 龙头涨跟风跌 |
| 8 | **窗口期追高位** | 4/8/10月暴雷季 |
| 9 | **量价背离不处理** | 虚假繁荣，主力出货 |
| 10 | **重仓单吊一只** | 黑天鹅直接归零 |

### 中危错误（影响收益）

| # | 错误 | 典型后果 |
|---|------|---------|
| 11 | **止损不执行** | 小亏变大亏 |
| 12 | **不止盈早卖** | 赚小钱丢大机会 |
| 13 | **盲目追热点** | 接最后一棒 |
| 14 | **频繁换股** | 手续费损耗 |
| 15 | **无仓位管理** | 满仓干，风险敞口大 |

## 📊 硬规则（Hard Rules）

> **违反以下任一规则，评级直接降为不可操作**

1. **风控扫描强制** — 推荐前必须 `python3 scripts/check_risk.py --code XXX`，安全⭐（极危险）禁止推荐
2. **T+1意识** — 当日买入不可卖出，隔夜仓不超过3成
3. **流动性优先** — 成交额<5000万的股票不作为主要标的
4. **日期严格区分** — "昨日涨停"≠"今日机会"，必须分开描述
5. **免责声明必须** — 所有报告结尾必须注明"仅供参考，不构成投资建议"
