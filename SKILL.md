## 🔀 核心流程（6步法）

```
Step 0: 理解需求 → 提问澄清（模糊需求必须澄清）
Step 1: 获取数据 → akShare/东方财富/TGB
Step 2: 多维分析 → 消息/情绪/技术/风险
Step 3: 四维评级 → 安全/机会/确定/舒适
Step 4: 输出结论 → 买入价+止损价+目标价+置信度
Step 5: 追踪反馈 → 复盘记录
```

### Step 0：需求澄清（必须）

> 模糊需求必须先澄清，不能直接给结论

| 用户说 | 必须澄清的问题 |
|--------|--------------|
| "有什么好股" | 见 [选股澄清清单](#选股澄清) |
| "能买吗XXX" | 见 [诊股澄清清单](#诊股澄清) |
| "我的要卖吗" | 持仓成本？持仓时长？ |
| "被套了" | 亏损幅度？仓位？ |

## 📋 输出规范（Anti-Slop）

> ❌ 避免空洞分析 ✅ 必须有具体内容

### 禁止出现的废话

| ❌ 禁止 | ✅ 正确 |
|--------|--------|
| "建议关注，风险自担" | 直接给操作建议 |
| "建议买入" | "建议买入，价格区间XX-XX，目标价XX" |
| "注意风险" | "止损价XX，跌破必须走" |
| 技术指标堆砌 | 只列关键指标 + 结论 |

### 必备内容

| # | 内容 | 示例 |
|---|------|------|
| 1 | **具体价位** | 买入价XX、止损价XX、目标价XX |
| 2 | **置信度** | 高/中/低 + 理由 |
| 3 | **核心逻辑** | 为什么现在买 |
| 4 | **时间窗口** | 验证周期（1-3日） |
| 5 | **仓位建议** | 首仓X成、最大X成 |

### 标准输出格式

```markdown
## [股票] 诊断报告 v1

**评级**：⭐⭐⭐⭐
**置信度**：中高
**核心逻辑**：[一句话]

| 参数 | 价位 | 说明 |
|------|------|------|
| 买入价 | XX.XX | 突破确认后介入 |
| 止损价 | XX.XX | 跌破必须走 |
| 目标价 | XX.XX | 止盈参考 |
| 仓位 | X成 | 不超过总仓位X成 |

**验证时间**：1-3日
**核心逻辑**：[为什么现在买]

*本报告仅供参考，不构成投资建议*
```

## 📈 TGB 情绪分析

> 淘股吧大V观点 + 散户情绪 = 真正的市场合力

### TGB 在四维中的位置

```
┌────────────────────────────────────────┐
│           情绪/资金维度（重点）           │
│                                         │
│   TGB情绪 ──────┐                        │
│   (散户合力)     │                        │
│        │        ▼                        │
│        │   游资动向 ──→ 龙虎榜/涨停板      │
│        │        │                        │
│        └──→ 主力资金 ──→ 北向/超大单       │
│                 │                        │
│                 ↓                        │
│           综合判断：时机是否成熟            │
└────────────────────────────────────────┘
```

### TGB 数据体系

| 数据源 | 内容 | 价值 |
|--------|------|------|
| **热度榜** | 股票讨论热度排名 | 提前发现异动 |
| **大V观点** | 影响力用户判断 | 参考但不盲从 |
| **实盘赛** | 真实收益率跟踪 | 验证判断准确率 |
| **帖子内容** | 散户多空观点 | 判断市场情绪顶底 |

### 大V观点整合

```
Step 1: 收集 → 热度排名 + 大V帖子 + 实盘赛
Step 2: 分类 → 看多(理由)/看空(理由)/中立(犹豫)
Step 3: 验证 → 逻辑是否与资金/技术面一致？
Step 4: 输出 → 主流观点 + 分歧点 + 综合判断
```

### TGB 情绪信号

| 信号 | 含义 | 操作 |
|------|------|------|
| 🟢 热度暴涨 + 大V唱多 + 技术突破 | 买入确认 | 积极 |
| 🟢 热度高 + 实盘赛重仓 | 趋势延续 | 持有 |
| 🟡 热度高 + 分歧大 | 震荡 | 高抛低吸 |
| 🔴 热度暴涨 + 大V提示风险 + 技术背离 | 警惕 | 减仓 |
| 🔴 极度悲观 + 无人讨论 | 情绪底部 | 观察不抄底 |

### 情绪顶底判断

```
情绪顶部（减仓信号）：
- 热度榜全是涨停股
- 大V普遍唱多
- "这次不一样"
- 新手蜂拥

情绪底部（观察机会）：
- 热度极低
- 大V普遍谨慎
- 市场完了言论
- 无人讨论
```

### TGB 分析输出格式

```markdown
### 情绪面（TGB）

**热度**：第X位 / 总讨论量XXX
**大V观点**：看多X人 / 看空X人 / 中立X人
**主流逻辑**：[核心唱多理由]
**分歧点**：[市场犹豫什么]

**情绪信号**：🟢高涨 / 🟡分化 / 🔴低迷

**综合判断**：
- 情绪 vs 技术：共振/背离
- 情绪 vs 资金：共振/背离
```

## 📊 硬规则

违反以下任一规则，评级直接降为「⭐ 不买」：

| # | 规则 |
|---|------|
| 1 | 风控扫描强制（R3红线） |
| 2 | 止损必须执行（7%） |
| 3 | T+1意识（当日买不可卖） |
| 4 | 流动性优先（成交额>5000万） |
| 5 | 仓位管理（单只≤30%，总≤50%） |
| 6 | 日期严格区分 |
| 7 | Disclaimer必须 |

python3 scripts/trading_plan.py --stock 300750
