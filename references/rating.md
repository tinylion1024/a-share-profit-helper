# 四维评级体系

> 四维度各1-5星，星级越高越好，星数代表等级而非评分

---

## 四维度定义

| 维度 | 含义 | 星级说明 |
|------|------|---------|
| **机会⭐** | 上涨空间与概率 | 1⭐=很小，5⭐=极大 |
| **安全⭐** | 风险程度（越稳越好） | 1⭐=高风险，5⭐=极安全 |
| **确定性⭐** | 上涨逻辑清晰度 | 1⭐=模糊，5⭐=非常确定 |
| **舒适度⭐** | 持有体验（位置/波动） | 1⭐=很不舒服，5⭐=很舒服 |

---

## 星级定义

### 机会维度

| 星级 | 含义 | 典型特征 |
|------|------|---------|
| ⭐ | 机会很小 | 震荡市、方向不明 |
| ⭐⭐ | 机会较小 | 反弹行情、空间有限 |
| ⭐⭐⭐ | 中等机会 | 趋势行情、正常空间 |
| ⭐⭐⭐⭐ | 机会较好 | 强势趋势、空间较大 |
| ⭐⭐⭐⭐⭐ | 绝佳机会 | 主升浪、催化剂明确 |

### 安全维度

| 星级 | 含义 | 典型特征 |
|------|------|---------|
| ⭐ | 高风险 | 退市/ST、流动性极差 |
| ⭐⭐ | 风险较大 | 高位股、业绩下滑 |
| ⭐⭐⭐ | 中等风险 | 正常股票、无明显风险 |
| ⭐⭐⭐⭐ | 较安全 | 行业龙头、业绩稳定 |
| ⭐⭐⭐⭐⭐ | 极安全 | 大盘蓝筹、政策支持 |

### 确定性维度

| 星级 | 含义 | 典型特征 |
|------|------|---------|
| ⭐ | 很模糊 | 纯概念、无实质支撑 |
| ⭐⭐ | 较模糊 | 消息不明确、联动差 |
| ⭐⭐⭐ | 中等确定 | 有催化剂但需验证 |
| ⭐⭐⭐⭐ | 较确定 | 多维度信号共振 |
| ⭐⭐⭐⭐⭐ | 非常确定 | 政策+业绩+技术共振 |

### 舒适度维度

| 星级 | 含义 | 典型特征 |
|------|------|---------|
| ⭐ | 很不舒服 | 高位追涨、成本附近 |
| ⭐⭐ | 较不舒服 | 小幅回调、波动大 |
| ⭐⭐⭐ | 中等舒适 | 正常位置、波动可控 |
| ⭐⭐⭐⭐ | 较舒服 | 回踩支撑、位置理想 |
| ⭐⭐⭐⭐⭐ | 很舒服 | 刚启动、回调充分 |

## 安全红线（R3红线）

> 安全⭐ < 3 时为高风险，禁止推荐

### R3红线项目（绝对禁止）

以下情况必须将安全⭐降至 1：

| # | 红线项目 | 说明 |
|---|---------|------|
| 1 | **立案调查** | 上市公司被证监会立案调查 |
| 2 | **退市风险** | *ST、**ST 股票 |
| 3 | **大比例减持** | 大股东/高管拟减持 > 5% |
| 4 | **业绩巨亏** | 季度/年报亏损超预期 |
| 5 | **窗口期高位** | 高位股在业绩窗口期默认 R3 |

### 业绩窗口期

| 时间窗口 | 风险处置 |
|---------|---------|
| 4月20日-30日 | 年报/一季报窗口，高位股安全⭐强制降至1 |
| 8月20日-31日 | 中报窗口，高位股安全⭐强制降至2 |
| 10月20日-31日 | 三季报窗口，高位股安全⭐强制降至2 |

## 评级决策树

```
问：能买吗？
  │
  ├─ 安全⭐ < 3 → ⭐ 不买（风险太大，R3红线）
  │
  ├─ 安全⭐ ≥ 3 → 继续评估
  │     │
  │     ├─ 机会⭐ ≥ 3 → 继续
  │     │     │
  │     │     ├─ 确定性⭐ ≥ 3 → 买/可以买
  │     │     └─ 确定性⭐ < 3 → 观望
  │     │
  │     └─ 机会⭐ < 3 → 不建议/观望
  │
  └─ 舒适度⭐ 可作为辅助参考
```

## 四维 → 操作映射

| 四维组合 | 操作建议 |
|---------|---------|
| 安全5 + 机会5 + 确定5 + 舒适5 | ⭐⭐⭐⭐⭐ 立即买，重仓 |
| 安全4 + 机会4 + 确定4 + 舒适4 | ⭐⭐⭐⭐ 可以买，正常参与 |
| 安全3 + 机会3 + 确定3 + 舒适3 | ⭐⭐⭐ 观望，等待确认 |
| 安全2 + 机会X + 确定X + 舒适X | ⭐⭐ 不建议，轻仓或不买 |
| 安全1 + 任意 | ⭐ 不买，禁止操作 |

## 综合评级计算

### 基础规则

- **安全⭐为门槛**：安全⭐ < 3 时，直接取消资格
- **加权方式**：机会(30%) + 安全(25%) + 确定(25%) + 舒适(20%)

### 计算公式

```
综合分 = 机会⭐×0.3 + 安全⭐×0.25 + 确定⭐×0.25 + 舒适⭐×0.2

综合评级：
- 4.5+  → ⭐⭐⭐⭐⭐ 立即买
- 3.5+  → ⭐⭐⭐⭐ 可以买
- 2.5+  → ⭐⭐⭐ 观望
- 1.5+  → ⭐⭐ 不建议
- <1.5  → ⭐ 不买
```

## 止损原则

| 止损类型 | 触发条件 | 操作 |
|---------|---------|------|
| **固定止损** | 亏损 7%-10% | 必须止损出局 |
| **移动止损** | 盈利 > 15% | 止损线上移至成本价 |
| **时间止损** | 持仓 > 5交易日无动静 | 考虑出局 |
