# 诊股

> 分析特定股票是否值得买入

## 触发词

- "能买XXX吗"
- "帮我看看XXX"
- "XXX怎么样"
- "XXX可以买入吗"

## 前置确认

- [ ] **确认分析日期**
- [ ] **确认最近交易日**（用于获取最新数据）
- [ ] 确认股票代码
- [ ] 确认持仓周期（短线/中线/长线）
- [ ] 确认买入预算/仓位

> 日期规则详见：[references/guides/date-guide.md](../../guides/date-guide.md)

## 分析步骤

### Step 1：环境自检

1. 确认 `MX_APIKEY` 和 `EM_API_KEY` 已加载
2. 运行 `market_analysis.py` 流动性扫描

### Step 2：R3风险扫描（必须先过）

运行 `check_risk.py --code [代码]`

```
R3红线触发 → 输出"不买"，结束
```

### Step 3：日期对齐

严禁混淆"昨日涨停/异动"与"今日实时表现"

### Step 4：三维看市场

判断大盘环境：

| 信号 | 环境 | 操作 |
|------|------|------|
| 🟢🟢🟢 | 强势 | 可积极买入 |
| 🟢🟡🟢 | 震荡偏强 | 精选时机 |
| 🔴🔴🔴 | 弱势 | 不买/观望 |

### Step 5：四维评级

| 维度 | 评估内容 |
|------|----------|
| 机会⭐ | 上涨空间、概率 |
| 安全⭐ | 风险程度、R3 |
| 确定⭐ | 上涨逻辑清晰度 |
| 舒适⭐ | 持有体验 |

### Step 6：买入舒适度判定

| 星级 | 情形 |
|------|------|
| ⭐⭐⭐⭐⭐ | 底部分歧转一致、缩量回踩均线、行业中军稳步放量 |
| ⭐⭐⭐ | 追涨非一字板龙头、板块内跟涨 |
| ⭐ | 一字板排单、高位放量分歧、业绩盲盒期 |

### Step 7：给出结论

```
安全⭐<3 → 不买
安全⭐≥3 + 机会⭐≥3 + 确定⭐≥3 → 可以买
```

## 供应链穿透（特殊题材）

面对断供/国产替代题材：

| 类型 | 特征 | 受益 |
|------|------|------|
| 致命缺货 | 短期无法替代 | 直接受益 |
| 认证期长 | 替代周期数年 | 受益有限 |

```markdown
## [股票名称] 诊断报告

**结论**：可以买 / 观望 / 不买
**三维**：消息🟢 情绪🟢 技术🟢
**四维**：机会X 安全X 确定X 舒适X
**综合分**：X.X

| 参数 | 价位 | 说明 |
|------|------|------|
| 买入价 | XX.XX | 突破确认后介入 |
| 止损价 | XX.XX | -7%必须走 |
| 目标价 | XX.XX | 分批止盈 |
| 仓位 | X成 | 不超总仓位X成 |

**核心逻辑**：XXX
**置信度**：高/中/低
**时间窗口**：X日内验证
```

## 硬规则检查

- ❌ R3红线触发 → 不买
- ❌ 止损>7%空间 → 降仓位
- ❌ T+1当日 → 不可买
