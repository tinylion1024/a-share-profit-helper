## SOP标准模板

每个SOP包含：

1. **触发词** — 用户可能问的典型问题
2. **前置确认** — 开始前需要收集的信息
3. **分析步骤** — 标准化的操作流程
4. **输出规范** — 报告的标准格式
5. **硬规则检查** — 必须遵守的规则
6. **关联SOP** — 相关场景指引
