# 每日复盘

> 复盘当日操作，对错分析，改进提升

## 触发词

- "我今天操作对吗"
- "帮我复盘"
- "今天操作怎么样"
- "操作反思"

## 前置确认

- [ ] **确认复盘日期**（用户指定 or 今日）
- [ ] **确认最近交易日**（用于核对数据）
- [ ] 确认操作记录（买/卖/持）
- [ ] 确认股票代码和价位

> 日期规则详见：[references/guides/date-guide.md](../../guides/date-guide.md)
