# 交易计划

> 每日盘后制定，为明日操作做好准备

## 工作流程

### Step 1：数据采集

- 上一交易日指标：成交量、资金流向、涨跌 → `mx-finance-search` 或 `mx-data`
- 15:00后V观点：情绪分析 → `taoguba-hot`

### Step 2：场景映射

| 场景 | 触发条件 | 应对 |
|------|----------|------|
| **乐观** | 龙头撬动、成交量放大、情绪回暖 | 积极参与主线 |
| **悲观** | 跌停扩散、亏钱效应、成交萎靡 | 防御/空仓 |

### Step 3：可执行锚点

| 要素 | 内容 |
|------|------|
| 买入价位 | 具体价格，突破确认后介入 |
| 止损价位 | 跌破必须走，7%硬止损 |
| 目标价位 | 分批止盈 |
| 成交量目标 | 放量确认/缩量预警 |

## 坑点

- ❌ 泛泛建议 → ✅ 具体股票代码作为锚点
- ❌ 模糊风控 → ✅ 明确"突破"或"破位"点
- ❌ 忽略日期 → ✅ 运行前验证是否为交易日
