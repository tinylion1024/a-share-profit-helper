# 盘后复盘

> 每个交易日 15:30 后完成，总结当日市场

## 四大组件

| 组件 | 内容 |
|------|------|
| **硬指标** | 指数表现、总成交量、主力净流向（按板块） |
| **情绪** | TGB热门关键词、V共识、"赚钱效应" vs "亏钱效应" |
| **领先指标** | 龙头股分析、周期阶段（启动/繁荣/混沌/退潮） |
| **板块轮动** | 聪明钱去向（如从科技转向公用事业） |

## 周期判断

| 阶段 | 特征 | 操作 |
|------|------|------|
| 启动期 | 龙头股首板、板块扩散 | 积极参与 |
| 繁荣期 | 连板股出现、情绪高潮 | 持有为主 |
| 混沌期 | 板块轮动快、龙头分歧 | 控制仓位 |
| 退潮期 | 跌停扩散、亏钱效应 | 休息/空仓 |

## 输出格式

```markdown
【复盘报告】
大盘：上证XX点 / 创业板XX点
成交额：XX亿（放量/缩量）
最强题材：XXX
龙头股：XXX
情绪：赚钱效应/亏钱效应扩散
明日关注：XXX
```
