# 四维分析法详解

> 四维度分析框架，详见 [SKILL.md 四维分析法](../SKILL.md#-四维分析法)

---

## 四维度定义

| 维度 | 分析内容 | 服务目标 |
|------|---------|---------|
| **消息/政策** | 宏观、行业、公司消息 | 找催化因素 |
| **情绪/资金** | 散户情绪、游资动向、资金流向 | 判断时机 |
| **流动性/技术** | 成交量、价格位置、均线 | 找切入点 |
| **风险/合规** | R3红线、业绩窗口、流动性 | 避免踩雷 |

---

## 维度详解

### 1. 消息/政策维度

**分析内容：**
- 宏观经济政策（货币政策、财政政策）
- 行业政策（扶持、限制、环保）
- 公司公告（业绩、并购、股权变动）
- 重大新闻（突发事件、行业动态）

**信号识别：**
| 信号 | 含义 | 操作 |
|------|------|------|
| 🟢 政策利好 | 顶层支持 | 积极关注 |
| 🟢 行业景气 | 板块机会 | 顺势而为 |
| 🟡 公司利好 | 需验证 | 谨慎乐观 |
| 🔴 政策利空 | 回避 | 减仓/不买 |
| 🔴 业绩地雷 | 回避 | 清仓 |

### 2. 情绪/资金维度

**分析内容：**
- 散户情绪（淘股吧热度、股吧讨论）
- 游资动向（龙虎榜、涨停板）
- 主力资金（北向资金、主力净流入）
- 板块联动（龙头带动、跟风效应）

**信号识别：**
| 信号 | 含义 | 操作 |
|------|------|------|
| 🟢 主力净流入 | 资金推动 | 积极关注 |
| 🟢 情绪高涨 | 趋势延续 | 顺势 |
| 🟡 板块轮动 | 节奏加快 | 高抛低吸 |
| 🔴 主力净流出 | 资金撤退 | 谨慎 |
| 🔴 情绪恐慌 | 底部信号 | 观察不抄底 |

### 3. 流动性/技术维度

**分析内容：**
- 成交量（缩量/放量、量比）
- 价格位置（高位/低位、均线位置）
- 技术形态（突破、回踩、整理）
- 趋势判断（上涨/下跌/震荡）

**信号识别：**
| 信号 | 含义 | 操作 |
|------|------|------|
| 🟢 放量突破 | 启动信号 | 买入 |
| 🟢 回踩支撑 | 二次机会 | 买入 |
| 🟡 缩量整理 | 等待突破 | 观望 |
| 🔴 放量下跌 | 出货信号 | 卖出 |
| 🔴 破位下跌 | 趋势破坏 | 止损 |

### 短中线选股优先级

当目标持仓周期在 **2-10 个交易日** 时，四维分析的排序应进一步收敛为：

1. **题材 / 催化优先**
   当前主线、扩散中的分支、确定性催化，要优先于静态便宜。
2. **趋势 / 结构优先**
   已经形成上升趋势、上方套牢盘较轻、回踩后仍有承接的票，优先于便宜但不涨的票。
3. **舒服买点优先**
   回踩支撑、突破回踩确认、缩量整理后再放量启动，优先于高潮追高。
4. **估值用于排序细化，不用于直接翻盘**
   低 PE / 低 PEG 只能作为加分项，不能直接压过更强的主线趋势股。

实战上要明确区分：

- **趋势票**
  可以在分歧承接、支撑回踩、确认再走强时参与。
- **性价比票**
  只有当题材开始轮动到它，或趋势出现明确修复，才提升优先级。
- **修复观察票**
  逻辑存在，但图形和筹码结构仍差，不应排在主线趋势票前面。

### 4. 风险/合规维度

**分析内容：**
- R3红线（见 rating.md）
- 业绩窗口期风险
- 流动性风险（日成交额）
- 退市风险（ST/*ST）

## 四维 → 操作决策映射

| 四维结论 | 赚钱引擎输出 | 学习引擎输出 |
|---------|------------|-------------|
| 消息积极 + 情绪高涨 + 技术突破 | → 买入信号 | → 理解为什么涨 |
| 消息利空 + 情绪恐慌 + 技术破位 | → 卖出/观望 | → 理解为什么跌 |
| 消息平淡 + 情绪稳定 + 技术整理 | → 观望等待 | → 理解市场平衡 |

## 四维分析流程

```
输入：股票代码/名称
  │
  ▼
┌─────────────────────────────────────────┐
│ 1. 消息/政策  →  找催化因素              │
│   - 政策是否利好？                       │
│   - 公司有无重大公告？                   │
│   - 催化剂是否明确？                    │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│ 2. 情绪/资金  →  判断时机                 │
│   - 主力资金动向？                       │
│   - 情绪热度？                           │
│   - 板块联动？                           │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│ 3. 流动性/技术  →  找切入点              │
│   - 是否突破？                           │
│   - 支撑位在哪？                         │
│   - 成交量配合？                         │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│ 4. 风险/合规  →  避免踩雷                │
│   - R3红线通过？                         │
│   - 业绩窗口？                           │
│   - 流动性足够？                         │
└─────────────────┬───────────────────────┘
                  ▼
           输出：四维评级 + 操作建议
```
