# 四维评级体系

> 四维度各1-5星，星级越高越好，星数代表等级而非评分

---

## 四维度索引

| 维度 | 文件 | 评估内容 |
|------|------|---------|
| **机会⭐** | [opportunity.md](opportunity.md) | 上涨空间与概率 |
| **安全⭐** | [safety.md](safety.md) | 风险程度（R3红线） |
| **确定性⭐** | [certainty.md](certainty.md) | 上涨逻辑清晰度 |
| **舒适度⭐** | [comfort.md](comfort.md) | 持有体验 |

---

## 快速决策

```
安全⭐ < 3 → ⭐ 不买（R3红线触发）

安全⭐ ≥ 3 → 继续评估
  │
  ├─ 机会⭐ ≥ 3 → 继续
  │     │
  │     └─ 确定性⭐ ≥ 3 → 买/可以买
  │
  └─ 机会⭐ < 3 → ⭐⭐ 观望/不建议
```

---

## 综合评级计算

### 加权公式

```
综合分 = 机会⭐×0.3 + 安全⭐×0.25 + 确定⭐×0.25 + 舒适⭐×0.2
```

### 综合评级

| 综合分 | 评级 | 操作建议 |
|--------|------|---------|
| 4.5+ | ⭐⭐⭐⭐⭐ | 立即买，重仓 |
| 3.5-4.4 | ⭐⭐⭐⭐ | 可以买，正常参与 |
| 2.5-3.4 | ⭐⭐⭐ | 观望，等待确认 |
| 1.5-2.4 | ⭐⭐ | 不建议，轻仓或不买 |
| <1.5 | ⭐ | 不买，禁止操作 |
