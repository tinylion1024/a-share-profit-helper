# 安全维度（Safety）

> 评估风险程度，星级越高越安全。<br>
> **⚠️ 安全⭐ < 3 为高风险，禁止推荐**

---

## 维度定义

| 属性 | 说明 |
|------|------|
| **含义** | 风险程度（越稳越好） |
| **星级说明** | 1⭐=高风险，5⭐=极安全 |
| **评估重点** | R3红线、基本面、流动性 |

---

## 评分规则

### 评分因素

| 因素 | 权重 | 说明 |
|------|------|------|
| **R3红线** | 40% | 是否有红线项目 |
| **基本面** | 25% | 业绩、估值、股东 |
| **流动性** | 20% | 日成交额 |
| **外部风险** | 15% | 政策、行业风险 |

### 评分标准

| 星级 | 含义 | 典型特征 | 评分要点 |
|------|------|---------|---------|
| ⭐ | 高风险 | 退市/ST、流动性极差 | R3红线触发，基本面恶化 |
| ⭐⭐ | 风险较大 | 高位股、业绩下滑 | 存在隐患，需谨慎 |
| ⭐⭐⭐ | 中等风险 | 正常股票、无明显风险 | 无明显风险，可接受 |
| ⭐⭐⭐⭐ | 较安全 | 行业龙头、业绩稳定 | 基本面良好，风险可控 |
| ⭐⭐⭐⭐⭐ | 极安全 | 大盘蓝筹、政策支持 | 基本面优异，风险极低 |

---

## R3红线（绝对禁止）

> 以下情况必须将安全⭐降至 1

| # | 红线项目 | 说明 | 自动评级 |
|---|---------|------|---------|
| 1 | **立案调查** | 上市公司被证监会立案调查 | ⭐ |
| 2 | **退市风险** | *ST、**ST 股票 | ⭐ |
| 3 | **大比例减持** | 大股东/高管拟减持 > 5% | ⭐ |
| 4 | **业绩巨亏** | 季度/年报亏损超预期 | ⭐ |
| 5 | **流动性枯竭** | 日成交额 < 1000万 | ⭐ |

---

## 评分检查清单

### ⭐ 高风险（1星）- R3红线

- [ ] 立案调查
- [ ] *ST/**ST
- [ ] 大股东减持>5%
- [ ] 业绩巨亏
- [ ] 流动性枯竭（日成交额<1000万）

### ⭐⭐ 风险较大（2星）

- [ ] 高位股（涨幅>50%）
- [ ] 业绩下滑
- [ ] 行业景气度下降
- [ ] 日成交额<5000万
- [ ] 窗口期高位股

### ⭐⭐⭐ 中等风险（3星）

- [ ] 正常股票
- [ ] 无明显风险事件
- [ ] 日成交额>5000万
- [ ] 业绩平稳

### ⭐⭐⭐⭐ 较安全（4星）

- [ ] 行业龙头
- [ ] 业绩稳定增长
- [ ] 日成交额>1亿
- [ ] 无大额解禁

### ⭐⭐⭐⭐⭐ 极安全（5星）

- [ ] 大盘蓝筹
- [ ] 政策支持
- [ ] 业绩优异
- [ ] 日成交额>5亿
- [ ] 机构重仓

---

## 业绩窗口期规则

| 时间窗口 | 高位股处理 |
|---------|-----------|
| 4月20日-30日 | 年报/一季报窗口，安全⭐强制降至1 |
| 8月20日-31日 | 中报窗口，安全⭐强制降至2 |
| 10月20日-31日 | 三季报窗口，安全⭐强制降至2 |

> **高位股定义**：近30日涨幅 > 30%

---

## 评分流程

```
Step 1: R3红线检查（必须首先检查）
  - 有红线 → 直接评定 ⭐，停止评分

Step 2: 基本面评估
  - 业绩稳定增长 → +2分
  - 业绩平稳 → +1分
  - 业绩下滑 → +0分

Step 3: 流动性评估
  - 日成交额>5亿 → +1分
  - 日成交额>1亿 → +0.5分
  - 日成交额<5000万 → +0分

Step 4: 外部风险
  - 政策支持 → +1分
  - 行业景气 → +0.5分
  - 行业下行 → +0分

Step 5: 综合评分
  合计4分以上 → 5星
  合计3分 → 4星
  合计2分 → 3星
  合计1分 → 2星
  R3触发 → 1星
```

---

## 场景示例

### 案例1：ST股

```
R3红线：退市风险 → 直接⭐
结论：⭐ 不买
```

### 案例2：行业龙头

```
R3红线：无 → 继续
基本面：业绩稳定 → +2分
流动性：日成交额8亿 → +1分
外部风险：政策支持 → +1分
合计：4分 → ⭐⭐⭐⭐
```

### 案例3：高位股+窗口期

```
时间：4月25日（年报窗口期）
近30日涨幅：40%（高位）
R3红线：窗口期高位 → 强制⭐
结论：⭐ 不买
```

## 安全维度 vs 其他维度关系

| 组合 | 评级影响 |
|------|---------|
| 安全⭐ < 3 | 直接取消资格，不考虑其他维度 |
| 安全⭐ ≥ 3 + 机会⭐ ≥ 3 | 可进入候选 |
| 安全⭐ ≥ 4 + 其他维度均衡 | 优质标的 |
