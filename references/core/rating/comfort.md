# 舒适度维度（Comfort）

> 评估持有体验（位置/波动），星级越高持有越舒服

---

## 维度定义

| 属性 | 说明 |
|------|------|
| **含义** | 持有体验（位置/波动） |
| **星级说明** | 1⭐=很不舒服，5⭐=很舒服 |
| **评估重点** | 当前位置、均线关系、波动性 |

---

## 评分规则

### 评分因素

| 因素 | 权重 | 说明 |
|------|------|------|
| **价格位置** | 35% | 相对历史高点的位置 |
| **均线关系** | 30% | 与均线的位置关系 |
| **波动性** | 20% | 振幅和波动大小 |
| **买入时机** | 15% | 相对启动点的位置 |

### 评分标准

| 星级 | 含义 | 典型特征 | 评分要点 |
|------|------|---------|---------|
| ⭐ | 很不舒服 | 高位追涨，成本附近 | 追高买入，均线远离 |
| ⭐⭐ | 较不舒服 | 小幅回调、波动大 | 上涨后回调，波动剧烈 |
| ⭐⭐⭐ | 中等舒适 | 正常位置、波动可控 | 正常位置，波动正常 |
| ⭐⭐⭐⭐ | 较舒服 | 回踩支撑、位置理想 | 回踩均线获得支撑 |
| ⭐⭐⭐⭐⭐ | 很舒服 | 刚启动、回调充分 | 启动初期，回调充分 |

---

## 评分检查清单

### ⭐ 很不舒服（1星）

- [ ] 高位追涨（相对近期高点<5%）
- [ ] 成本在高位
- [ ] 均线完全发散

### ⭐⭐ 较不舒服（2星）

- [ ] 小幅回调（高位>5%但<15%）
- [ ] 波动大（振幅>10%）
- [ ] 刚错过最佳买点

### ⭐⭐⭐ 中等舒适（3星）

- [ ] 正常位置（中等区间）
- [ ] 波动正常（振幅5-10%）
- [ ] 均线纠缠

### ⭐⭐⭐⭐ 较舒服（4星）

- [ ] 回踩均线获得支撑
- [ ] 价格在均线上方不远处
- [ ] 波动收敛（振幅3-5%）

### ⭐⭐⭐⭐⭐ 很舒服（5星）

- [ ] 刚启动（启动<10%）
- [ ] 回调充分（回踩重要均线）
- [ ] 低波动（振幅<3%）
- [ ] 均线多头排列刚形成

---

## 位置评估标准

### 价格相对位置

| 位置 | 说明 | 评分 |
|------|------|------|
| 近30日高点 < 5% | 高位 | +0分 |
| 近30日高点 5-15% | 中高位 | +0.5分 |
| 近30日高点 15-30% | 中位 | +1分 |
| 近30日高点 30-50% | 中低位 | +1.5分 |
| 近30日高点 > 50% | 低位/刚启动 | +2分 |

### 均线关系

| 关系 | 说明 | 评分 |
|------|------|------|
| 价格 < MA5 < MA10 < MA20 | 空头排列 | +0分 |
| 价格在均线附近纠缠 | 震荡 | +0.5分 |
| MA5 > MA10 > MA20，但价格回踩 | 多头回踩 | +1.5分 |
| 价格 > MA5 > MA10 > MA20 | 标准多头 | +1分 |
| 价格刚突破所有均线 | 刚突破 | +2分 |

---

## 评分流程

```
Step 1: 价格位置评估
  - 低位/刚启动（>50%距离高点）→ +2分
  - 中低位（30-50%）→ +1.5分
  - 中位（15-30%）→ +1分
  - 中高位（5-15%）→ +0.5分
  - 高位（<5%）→ +0分

Step 2: 均线关系评估
  - 刚突破所有均线 → +2分
  - 多头回踩均线 → +1.5分
  - 标准多头排列 → +1分
  - 均线纠缠 → +0.5分
  - 空头排列 → +0分

Step 3: 波动性评估
  - 低波动（振幅<3%）→ +1分
  - 正常波动（3-5%）→ +0.5分
  - 高波动（>5%）→ +0分

Step 4: 买入时机评估
  - 启动<10% → +1分
  - 启动10-20% → +0.5分
  - 启动>20% → +0分

Step 5: 综合评分
  合计5分以上 → 5星
  合计4分 → 4星
  合计2-3分 → 3星
  合计1分 → 2星
  合计0分 → 1星
```

## 场景示例

### 案例1：刚启动龙头

```
价格位置：启动8% → +2分
均线关系：刚突破所有均线 → +2分
波动性：振幅2% → +1分
买入时机：启动初期 → +1分
合计：6分 → ⭐⭐⭐⭐⭐
```

### 案例2：高位追涨

```
价格位置：距高点3% → +0分
均线关系：均线发散 → +0分
波动性：振幅12% → +0分
买入时机：已涨30% → +0分
合计：0分 → ⭐
```

### 案例3：回踩均线

```
价格位置：高位15% → +0.5分
均线关系：回踩MA20获得支撑 → +1.5分
波动性：振幅4% → +0.5分
买入时机：启动25% → +0分
合计：2.5分 → ⭐⭐⭐
```

## 舒适度维度 vs 其他维度关系

| 组合 | 评级影响 |
|------|---------|
| 舒适度⭐ = 1 | 持有体验差，可能影响执行 |
| 舒适度⭐ = 5 | 最佳买入时机 |
| 舒适度高 + 确定高 | 双重确认 |
| 舒适度低 + 机会高 | 可接受但需更强确定性 |
| 舒适度低 + 安全低 | 明确不买 |

---

## 舒适度辅助说明

> 舒适度是辅助维度，不单独决定买卖，但影响持仓心态

| 场景 | 舒适度处理 |
|------|----------|
| 舒适度⭐=5 | 可适当放大仓位 |
| 舒适度⭐=3 | 标准仓位 |
| 舒适度⭐=1 | 降低仓位或观望 |
