## 大V体系

### 顶级大V（粉丝>50万）

| 特征 | 影响 | 应对 |
|------|------|------|
| 号召力强 | 一呼百应 | 关注但不盲从 |
| 追随者众 | 容易形成合力 | 观察跟风效应 |
| 观点鲜明 | 操作参考价值高 | 验证逻辑 |

### 识别要点

```
大V质量判断：
1. 是否有持续实盘记录？
2. 逻辑是否清晰可验证？
3. 历史判断准确率？
4. 是否与主流机构观点一致？
```

## TGB 情绪指标

### 热度指标

| 指标 | 含义 | 信号 |
|------|------|------|
| **热度排名** | 个股讨论热度 | 热度突然提升 → 关注 |
| **帖子数量** | 短时间内新增帖子 | 爆发式增长 → 警惕 |
| **阅读量** | 帖子总阅读数 | 爆款帖子 → 注意 |

### 情绪指标

| 指标 | 含义 | 信号 |
|------|------|------|
| **多空比例** | 看多vs看空 | 多空比>3:1 → 谨慎 |
| **情绪极值** | 极度乐观/悲观 | 情绪顶/底部 → 反转信号 |
| **话题集中度** | 讨论是否集中某股 | 过度集中 → 警惕 |

## TGB 情绪 → 操作映射

### 情绪信号

| TGB情绪 | 技术信号 | 操作 |
|---------|---------|------|
| 热度暴涨 + 大V唱多 | 配合突破 | 买入确认 |
| 热度高 + 分歧大 | 震荡 | 高抛低吸 |
| 热度低 + 无人讨论 | 盘整/冷门 | 等待 |
| 热度暴涨 + 大V提示风险 | 配合出货 | 卖出/观望 |
| 极度悲观 + 无人看多 | 情绪底部 | 观察抄底机会 |

### 情绪顶底判断

```
情绪顶部特征：
- 热度榜全是涨停股
- 大V普遍唱多
- "这次不一样"言论
- 新手蜂拥而入
→ 减仓信号

情绪底部特征：
- 热度极低
- 大V普遍谨慎
- "市场完了"言论
- 无人讨论
→ 观察机会
```

## 大V观点整合流程

```
Step 1: 收集大V观点
  - 热度排名靠前的帖子
  - 粉丝>10万的大V
  - 有实盘记录的用户

Step 2: 观点分类
  - 看多（理由是什么）
  - 看空（理由是什么）
  - 中立（犹豫什么）

Step 3: 验证逻辑
  - 逻辑是否与四维分析一致？
  - 是否与主力动向一致？
  - 历史判断准确率？

Step 4: 整合输出
  - 主流观点是什么？
  - 分歧点在哪里？
  - 综合判断：机会 vs 风险
```

## TGB 与其他维度联动

### 消息面联动

| TGB热议 | 消息面 | 解读 |
|---------|--------|------|
| 突然热议某股 | 利好公告前 | 资金先知先觉 |

| 热议某概念 | 无消息 | 主力试探？ |

| TGB情绪 | 技术面 | 解读 |
|---------|--------|------|
| 唱多 + 突破 | 形成 | 买入确认 |
| 唱多 + 背离 | 顶背离 | 警惕 |
| 看空 + 破位 | 形成 | 卖出信号 |
| 看空 + 底背离 | 形成 | 观察机会 |

### 资金面联动

| TGB情绪 | 资金面 | 解读 |
|---------|--------|------|
| 唱多 + 主力净流入 | 形成 | 跟进 |
| 唱多 + 主力净流出 | 背离 | 警惕 |
| 看空 + 主力净流出 | 形成 | 回避 |
| 看空 + 主力净流入 | 背离 | 可能反转 |

## TGB 情绪分析实战

### 案例：龙头股

```
TGB观察：
- 热度榜前3
- 大V普遍唱多
- 实盘赛选手重仓

分析：
- 情绪高涨 → 趋势延续
- 但需警惕 → 情绪顶
- 验证技术面 → 是否加速？

结论：
- 可持有，但设移动止损
- 不追高
- 关注TGB情绪变化
```

### 案例：超跌股

```
TGB观察：
- 热度极低
- 无人讨论
- 大V普遍悲观

分析：
- 情绪底部特征
- 但可能继续跌
- 等待技术面企稳

结论：
- 不抄底
- 等待情绪好转
- 等待技术面信号
```

## TGB 情绪避坑

### 大V依赖症

```
❌ 错误：跟着大V操作，不自己判断
✅ 正确：大V观点是参考，必须独立验证
```

### 情绪滞后

```
❌ 错误：看到热议才去买
✅ 正确：TGB情绪反映的是过去，应预判
```

### 以偏概全

```
❌ 错误：只看唱多的帖子
✅ 正确：多空帖子都要看，分析分歧
```

### 情绪顶底判断

```
❌ 错误：情绪顶仍看多，情绪底仍看空
✅ 正确：情绪极值是反向指标
```

```markdown
### 情绪面（TGB）

**热度**：第X位 / 总讨论量XXX
**大V观点**：看多X人 / 看空X人 / 中立X人
**主流逻辑**：[核心唱多理由]
**分歧点**：[市场犹豫什么]

**情绪信号**：🟢高涨 / 🟡分化 / 🔴低迷

**综合判断**：
- 情绪面 vs 技术面：共振/背离
- 情绪面 vs 资金面：共振/背离
```

## 配置项

> 在 config.json 中配置TGB分析偏好

```json
{
  "sentiment": {
    "tgbot_threshold": 1000,
    "top_vuser_fans": 100000,
    "include实盘赛": true,
    "include热帖": true
  }
}
```

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| tgbot_threshold | 热议门槛（帖子数） | 1000 |
| top_vuser_fans | 大V粉丝门槛 | 10万 |
| include实盘赛 | 是否包含实盘赛数据 | true |
| include热帖 | 是否包含热帖分析 | true |
