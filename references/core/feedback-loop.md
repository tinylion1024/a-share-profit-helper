# 反馈循环（Feedback Loop）详解

> 持续进化机制，详见 [SKILL.md 反馈循环](../SKILL.md#-反馈循环)

---

## 循环流程

```
┌─────────────────────────────────────────────────────────────┐
│                       Feedback Loop                          │
│                                                             │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐          │
│   │  操作执行 │ → │ 结果收集 │ → │ 学习更新 │          │
│   └─────┬────┘    └─────┬────┘    └─────┬────┘          │
│         ↑               ↓               ↓                   │
│         │          ┌──────────┐    ┌──────────┐          │
│         └──────── │  案例积累 │ → │ 规则优化 │          │
│                    └──────────┘    └──────────┘          │
└─────────────────────────────────────────────────────────────┘
```

---

## 反馈收集类型

| 类型 | 收集内容 | 触发时机 |
|------|---------|---------|
| **操作反馈** | 买入/卖出价格、时间、结果 | 每次操作后 |
| **评级反馈** | 评级 vs 实际走势 | 持仓变化时 |
| **预测反馈** | 预测方向 vs 实际结果 | 大盘前瞻验证时 |
| **学习反馈** | 理解程度、操作改进 | 复盘完成后 |

---

## 进化触发机制

| 触发条件 | 进化内容 | 优先级 |
|---------|---------|--------|
| 每次操作完成 | 记录案例 | 高 |
| 评级连续偏差3次 | 调整评级权重 | 高 |
| 发现新市场规律 | 更新分析维度 | 中 |
| 高胜率策略发现 | 加入推荐规则 | 中 |
| 重大错误发生 | 紧急更新 | 高 |

## 案例库结构

```
references/cases/
├── 成功案例/
│   ├── 选股成功/      # 选对股票的案例
│   ├── 止损成功/      # 及时止损减少亏损
│   ├── 抄底成功/      # 成功买在低位
│   └── 复利增长/      # 稳定盈利的记录
├── 失败案例/
│   ├── 追高失败/      # 追涨被套
│   ├── 止损不及/      # 止损太慢
│   ├── 踩雷/         # 踩雷退市
│   └── 割肉/         # 不该割时割了
└── 市场规律/
    ├── 板块轮动/      # 板块轮动规律
    ├── 情绪周期/      # 情绪转折规律
    └── 资金流向/      # 资金运动规律
```

## 规律更新流程

```
案例积累 ≥ 10个
     │
     ▼
┌─────────────────────────────────┐
│  归因分析                       │
│  - 成功案例共同特征？           │
│  - 失败案例共同错误？           │
│  - 评级偏差规律？               │
└─────────────────┬───────────────┘
                  ▼
┌─────────────────────────────────┐
│  规则更新                       │
│  - 新增筛选条件？               │
│  - 调整评级权重？               │
│  - 更新操作锚点？               │
└─────────────────┬───────────────┘
                  ▼
┌─────────────────────────────────┐
│  验证测试                       │
│  - 回测新规则效果               │
│  - 如有效则正式更新             │
└─────────────────────────────────┘
```

---

## 进化引擎工作流

```
日常反馈
    │
    ├─→ 每次操作 → 记录案例
    │
    ├─→ 每日复盘 → 分析对错
    │
    └─→ 每周总结 → 规律更新

定期进化
    │
    ├─→ 月度 → 案例库回顾
    │
    ├─→ 季度 → 规则体系优化
    │
    └─→ 年度 → 模型参数调整
```

## 反馈质量标准

### 操作反馈质量

| 字段 | 必须 | 可选 |
|------|------|------|
| 买入日期/价格 | ✅ | |
| 卖出日期/价格 | ✅ | |
| 四维评级 | ✅ | |
| 操作理由 | ✅ | |
| 实际走势 | ✅ | |
| 误差分析 | | ✅ |
| 改进建议 | | ✅ |

### 评级反馈质量

| 字段 | 说明 |
|------|------|
| 评级 vs 实际 | 方向是否正确 |
| 幅度偏差 | 预测 vs 实际涨跌幅 |
| 时间偏差 | 预测 vs 实际启动时间 |
| 偏差原因 | 哪些因素未考虑 |
