## 买入锚点

| 参数 | 确定方法 | 示例 |
|------|---------|------|
| **买入价** | 突破确认价 + 2% | 突破10元 → 10.2元买 |
| **首仓** | 总资金10% | 100万 → 首仓10万 |
| **加仓** | 首次盈利后+5% | 盈利5%后加仓 |
| **止损** | 买入价下方7% | 10.2买 → 9.5止损 |

### 买入时机

```
最佳买点：
1. 回踩支撑时买入（MA5/MA10/MA20）
2. 突破后回踩确认时买入
3. 缩量整理后放量启动时买入

禁止买点：
1. 连续大涨后追高
2. 一字板后买入
3. 尾盘集合竞价买入（次日T+1风险）
```

### 买入流程

```
1. 确认四维评级（安全≥3, 机会≥3）
2. 确定买入价（突破价+2%）
3. 计算止损价（买入价-7%）
4. 计算目标价（止损距离×3）
5. 确定首仓比例（10%）
6. 执行买入
```

## 卖出锚点

| 参数 | 确定方法 | 示例 |
|------|---------|------|
| **目标价** | 止损距离×3 | 止损0.7元 → 目标2.1元 |
| **止盈线** | 成本价+10% | 10.2买 → 11.2考虑止盈 |
| **分批卖** | 目标50%卖一半 | 到11元卖一半 |

### 卖出时机

```
分批卖出：
1. 达到目标价 → 卖50%
2. 继续上涨10% → 再卖30%
3. 剩余20%移动止损

禁止卖出：
1. 小幅波动止损
2. 到达目标后贪婪不止盈
3. 下跌时不执行止损
```

### 卖出流程

```
1. 达到目标价 → 执行卖出50%
2. 移动止损线上移（成本价）
3. 继续上涨 → 分批卖出
4. 反转下跌 → 执行止损
```

## 仓位模型

| 市场环境 | 总仓位 | 单只上限 |
|---------|-------|---------|
| 大盘上涨趋势 | 5成仓 | 2成 |
| 大盘震荡 | 3成仓 | 1成 |
| 大盘下跌 | 1成仓 | 0.5成 |
| 强势机会 | +1成仓 | +0.5成 |

### 仓位计算

```
总资金：100万

大盘上涨趋势（5成仓 = 50万）：
  - 单只上限：20万（2成）
  - 可买2-3只

大盘震荡（3成仓 = 30万）：
  - 单只上限：10万（1成）
  - 可买3只

大盘下跌（1成仓 = 10万）：
  - 单只上限：5万（0.5成）
  - 只做短线
```

## 止损原则

| 类型 | 条件 | 操作 |
|------|------|------|
| **固定止损** | 亏损 7%-10% | 必须止损 |
| **移动止损** | 盈利 > 15% | 线上移至成本 |
| **时间止损** | 持仓 > 5日无动静 | 考虑出局 |

### 止损执行

```
亏损7%时：
  - 立即执行卖出
  - 不犹豫、不补仓
  - 记录亏损原因

盈利15%后：
  - 止损线移至成本价
  - 锁定部分利润
  - 让利润奔跑
```

## T+1 规则

> A股实行T+1，当日买入不可卖出

### 隔夜仓规则

```
收盘前检查：
- 今日买入股 → 不可卖
- 昨日买入股 → 可卖
- 持仓不超过总资金30%
```

### 典型错误

| 错误 | 后果 |
|------|------|
| 当日买卖同一股 | 违规无法成交 |
| 隔夜满仓 | 次日无法加仓/补救 |
| 追涨停板 | 次日低开被套 |
