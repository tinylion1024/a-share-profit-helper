# 反模式详解

> 散户常见亏损行为避坑指南，详见 [SKILL.md 反模式](../SKILL.md#-反模式)

---

## 致命错误（必死无疑）

| # | 错误 | 典型后果 | 避免方法 |
|---|------|---------|---------|
| 1 | **安全⭐仍买入** | 被套、踩雷、退市 | 先过风控扫描 |
| 2 | **不止损** | 小亏变大亏，最终腰斩 | 7%必须走 |
| 3 | **高位追一字板** | 次日低开-15%起步 | 不追涨停板 |
| 4 | **重仓单吊** | 黑天鹅直接归零 | 单只不超3成 |
| 5 | **T+1当天买卖** | 违规无法成交 | 当日买不可卖 |

---

## 高危错误（大概率亏损）

| # | 错误 | 典型后果 | 避免方法 |
|---|------|---------|---------|
| 6 | **逆势抄底** | 接飞刀、越套越深 | 顺势而为 |
| 7 | **消息兑现追涨** | 利好出尽变利空 | 利好前潜伏 |
| 8 | **忽视板块联动** | 龙头涨跟风跌 | 看板块整体 |
| 9 | **窗口期追高位** | 4/8/10月暴雷季 | 窗口期降仓 |
| 10 | **不止盈** | 赚变亏，坐过山车 | 目标价分批卖 |

## 中危错误（影响收益）

| # | 错误 | 典型后果 | 避免方法 |
|---|------|---------|---------|
| 11 | **频繁换股** | 手续费损耗 | 减少交易频率 |
| 12 | **仓位失控** | 满仓干，风险敞口大 | 严格仓位管理 |
| 13 | **盲目追热点** | 接最后一棒 | 热点持续性判断 |
| 14 | **听消息炒股** | 消息往往滞后 | 以我为主 |
| 15 | **不记录操作** | 同样的错误重复犯 | 建立交易日志 |

## 反模式分类记忆

### 必死的五条

```
安全⭐<3 仍买入 → 踩雷退市
不止损       → 腰斩
追一字板     → -15%起步
重仓单吊     → 归零
T+1当天买卖  → 违规
```

### 大概率亏的五条

```
逆势抄底     → 接飞刀
消息兑现追涨 → 利空
忽视联动     → 跟风跌
窗口期高位   → 暴雷
不止盈       → 坐过山车
```

### 影响收益的五条

```
频繁换股     → 手续费
仓位失控     → 风险大
盲目追热点   → 接盘
听消息炒股   → 滞后
不记录操作   → 重复错
```

## 亏损原因归因

### 心态问题

| 问题 | 表现 | 解决方案 |
|------|------|---------|
| 贪婪 | 不止盈、满仓干 | 机械执行目标价 |
| 恐惧 | 提前卖、止损太严 | 放宽止损至7% |
| 侥幸 | 不止损、越跌越买 | 7%止损不犹豫 |
| 从众 | 追热点、听消息 | 独立判断 |

### 认知问题

| 问题 | 表现 | 解决方案 |
|------|------|---------|
| 过度自信 | 重仓、频繁交易 | 降仓至1成 |
| 确认偏差 | 只看利好 | 四维分析 |
| 损失厌恶 | 不止损 | 止损即盈利 |
| 锚定效应 | 成本价执念 | 看未来不看成本 |

### 执行问题

| 问题 | 表现 | 解决方案 |
|------|------|---------|
| 知行不一 | 知道不止损 | 设自动警报 |
| 纪律缺失 | 临时改计划 | 交易计划书面化 |
| 借口太多 | 下跌找理由 | 止损后不复盘 |

## 避坑检查清单

### 买入前检查

- [ ] 安全⭐≥3
- [ ] R3红线未触发
- [ ] 止损价已设定
- [ ] 仓位不超3成
- [ ] 非高位追涨

### 持仓中检查

- [ ] 跌幅<7%？
- [ ] 非T+1违规？
- [ ] 非窗口期暴雷？
- [ ] 板块联动正常？

### 卖出后检查

- [ ] 执行了止损？
- [ ] 达到目标分批卖？
- [ ] 记录了案例？
- [ ] 分析了对错？

## 典型错误案例

### 案例1：安全⭐=1仍买入

```
错误：*ST股安全⭐=1仍买入
结果：退市，资产归零
教训：安全⭐<3 绝对不买
```

### 案例2：不止损

```
错误：亏损20%不止损
结果：腰斩再腰斩
教训：7%必须走，不侥幸
```

### 案例3：高位追一字板

```
错误：涨停次日追涨停
结果：低开-15%
教训：一字板不追，尾盘不买
```

### 案例4：重仓单吊

```
错误：100万单吊一只
结果：黑天鹅，资产归零
教训：单只不超3成
```

### 案例5：逆势抄底

```
错误：大盘下跌抄底
结果：接飞刀，被套50%
教训：顺势而为，不接刀
```

## A股常见陷阱（补充）

### 一字板陷阱

| 情形 | 风险 | 应对 |
|------|------|------|
| 一字板涨停次日高开 | 往往低走 | 观望，不追 |
| 一字板涨停当日排单 | 买不进 | 等炸板回封 |
| 连续一字板后首次开板 | 可能是顶部 | 坚决卖出 |

### 高位放量分歧

| 信号 | 含义 | 操作 |
|------|------|------|
| 高位放量大阴线 | 主力出货 | 坚决卖出 |
| 高位放量十字星 | 多空分歧 | 观望 |
| 高位缩量盘整 | 主力控盘 | 谨慎 |

### 流动性陷阱

| 情形 | 风险 | 应对 |
|------|------|------|
| 冷门股 | 日成交额<5000万，流动性差 | 优先选成交额排名前100 |
| 小票 | 市值<50亿，易被操纵 | 谨慎 |

> 与上方「反模式」配合使用，形成完整的避坑体系
