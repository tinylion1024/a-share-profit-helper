# 日期指南

> 规范日期判断，避免时间混淆

---

## 核心概念

### 最近交易日（Last Trading Day）

最近交易日 = 当前日期前最后一个A股交易日

**计算规则：**
- 周一~周五（非节假日）= 交易日
- 周六、周日 = 非交易日
- 法定节假日 = 非交易日

**节假日判断：**
- 春节（除夕前1天 ~ 正月初7）
- 国庆（10月1日 ~ 10月7日）
- 其他法定节假日

### 举例：

| 当前日期 | 最近交易日 |
|----------|------------|
| 2026-04-27(周一) | 2026-04-27 |
| 2026-04-26(周日) | 2026-04-24(周五) |
| 2026-02-16(春节后周一) | 2026-02-16 |
| 2026-02-17(春节后周二) | 2026-02-17 |

## 时间段判断

### 自动判断流程

```
当前时间判断：
1. 今天是交易日？
   - 是 → 判断具体时段
   - 否 → 最近交易日 = 昨天/上周五
```

### 时段与操作

| 时段 | 时间 | 推荐操作 |
|------|------|----------|
| 盘前 | 08:00前 | 复盘昨日，做今日计划 |
| 竞价 | 09:15-09:25 | 观察竞价信号 |
| 盘中 | 09:30-11:30 / 13:00-15:00 | 谨慎操作，关注信号 |
| 盘后 | 15:00后 | 做收盘复盘 |
| 晚间 | 18:00后 | 做明日计划 |

## 各场景日期处理

### 场景1：周日想复盘

**错误做法：** 只看"上周五"数据

**正确做法：**
```
当前：周日(4/26)
最近交易日：周五(4/24)
应分析：周五收盘数据 + 周末消息
```

### 场景2：节假日前最后一个交易日

**正确做法：**
```
春节前最后一个交易日：2月14日
应分析：2月14日收盘数据
并考虑：长假期间外盘涨跌、消息面变化
```

### 场景3：节后第一个交易日

**正确做法：**
```
春节后第一个交易日：2月17日
应分析：2月17日开盘数据
并考虑：节日期间积累的消息、外盘表现
```

### 场景4：T+1计算

```
周一买 → 周二可卖
周五买 → 下周一可卖
周四买 → 周五可卖（不过周末）

注意：T日买的股票，T日不能卖
```

## SOP日期要求

每个SOP前置确认必须包含：

- [ ] 确认分析日期（用户指定 vs 当前日期）
- [ ] 确认最近交易日
- [ ] 确认时段（盘前/盘中/盘后）

---

## 日期标注规范

所有输出必须标注：

```
【分析日期】YYYY-MM-DD
【最近交易日】YYYY-MM-DD
【数据时间】YYYY-MM-DD HH:MM:SS
```

## 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 周日说"上周五" | 数据过时 | 用最近交易日 |
| 混淆"昨日" | 以为是昨天 | 明确是上一个交易日 |
| T+1算错 | 违规无法成交 | 下一交易日才能卖 |
| 节假日后第一个交易日当平时 | 忽略长假消息 | 考虑节日积累信息 |
