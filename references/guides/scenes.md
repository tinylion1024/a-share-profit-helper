# 场景体系详解

> 15个核心场景，详见 [SKILL.md 场景体系](../SKILL.md#-场景体系)

## A. 赚钱引擎（6场景）

| 场景 | 散户问 | 系统输出 | 反馈点 |
|------|---------|---------|--------|
| **选股** | "什么能让我赚钱？" | 机会清单 + 理由 + 预期收益 | 操作结果 |
| **诊股** | "我能买XXX吗？" | 买/观望/不买 + 价位 + 仓位 | 涨跌结果 |
| **持仓诊断** | "我的要卖吗？" | 持有/加/减/清 + 价位 | 操作结果 |
| **解套** | "被套了怎么办？" | 躺平/补仓/割肉/换股 + 时间线 | 解套结果 |
| **止损** | "亏多少必须走？" | 止损价 + 纪律提醒 | 是否执行 |
| **止盈** | "赚多少要跑？" | 目标价 + 分批策略 | 是否执行 |

## B. 学习引擎（6场景）

| 场景 | 散户问 | 系统输出 | 反馈点 |
|------|---------|---------|--------|
| **大盘前瞻** | "今天怎么看？" | 方向 + 理由 + 仓位建议 | 实际走势 |
| **大盘总结** | "今天为什么涨/跌？" | 归因 + 规律 + 明日展望 | 明日验证 |
| **每日复盘** | "我今天操作对吗？" | 对错 + 原因 + 改进点 | 认知提升 |
| **案例拆解** | "这个股票为什么涨？" | 深层逻辑 + 经验 | 知识积累 |
| **错误反思** | "我为什么会亏？" | 归因 + 改进方案 | 避免重复 |
| **经验沉淀** | "我怎么赚到的？" | 成功逻辑 + 可复制性 | 复制成功 |

## C. 进化引擎（3场景）

| 场景 | 输入 | 输出 | 反馈点 |
|------|------|------|--------|
| **案例记录** | 操作结果 | 归档案例库 | 持续积累 |
| **规律更新** | 案例分析 | 更新分析规则 | 能力提升 |
| **模型优化** | 历史数据 | 调整参数/权重 | 准确性提升 |
