# 硬规则

> 违反以下任一规则，评级直接降为「⭐ 不买」

---

## 七条铁律

| # | 规则 | 原因 | 后果 |
|---|------|------|------|
| 1 | **风控扫描强制** | R3红线未过必须跳过，安全⭐即禁止推荐 | ⭐ 不买 |
| 2 | **止损必须执行** | 不执行止损是散户亏损的主要原因 | 7%必须走 |
| 3 | **T+1意识** | 当日买不可卖，隔夜仓不超3成 | 当日买不可卖 |
| 4 | **流动性优先** | 成交额<5000万股票不作为主要标的 | 避开小票 |
| 5 | **仓位管理** | 单只不超3成仓，总仓位不超过5成 | 分散风险 |
| 6 | **日期严格区分** | "昨日涨停"≠"今日机会"，必须分开描述 | 不混淆 |
| 7 | **Disclaimer必须** | 所有报告必须注明"仅供参考，不构成投资建议" | 合规 |

---

## 规则详解

### 规则1：风控扫描强制

```
买入前必须通过 R3 红线扫描：
- 立案调查？→ R3
- *ST/**ST？→ R3
- 大股东减持>5%？→ R3
- 业绩巨亏？→ R3
- 窗口期高位股？→ R3

安全⭐ < 3 → 直接不买
```

### 规则2：止损必须执行

```
7% 止损原则：
- 亏损达到7% → 必须止损，不犹豫
- 不补仓摊薄成本
- 不侥幸等待反弹
- 记录亏损原因
```

### 规则3：T+1意识

```
A股 T+1 规则：
- 当日买入的股票，当日不可卖出
- 隔夜仓不超过总仓位30%

操作注意：
- 尾盘买，不隔夜
- 追涨停板次日低开风险
```

### 规则4：流动性优先

```
最小成交额：5000万/日

低流动性风险：
- 买入难以成交
- 卖出难以成交
- 容易被操纵

成交额筛选：
- 日成交额 < 5000万 → 不作为主要标的
- 日成交额 > 1亿 → 优先考虑
```

### 规则5：仓位管理

```
仓位限制：
- 单只股票：最大 30%
- 总仓位：最大 50%（平衡型）

仓位计算：
- 100万资金
- 单只上限 = 30万
- 总仓位上限 = 50万
- 可买 1-2 只
```

### 规则6：日期严格区分

```
易错点：
- "昨日涨停" ≠ "今日机会"
- "历史高点" ≠ "当前阻力"
- "曾经大跌" ≠ "现在是底"

必须：
- 分清"历史"和"现在"
- 单独分析每个时间点
- 不拿昨日数据直接套今天
```

### 规则7：Disclaimer必须

```
所有输出必须包含：

---
*本报告仅供参考，不构成投资建议*
*据此操作，盈亏自负*
---
```

## 快速检查清单

### 买入前检查

- [ ] 风控扫描通过（安全⭐≥3）
- [ ] R3红线无触发
- [ ] 成交额 > 5000万
- [ ] 单只仓位 ≤ 30%
- [ ] 总仓位 ≤ 50%
- [ ] 非T+1违规
- [ ] 有止损价
- [ ] Disclaimer已加

### 持仓中检查

- [ ] 亏损 < 7%？
- [ ] 遵守T+1规则？
- [ ] 仓位合规？

### 卖出后检查

- [ ] 执行了止损？
- [ ] 达到目标分批卖？
- [ ] 记录了案例？
- [ ] 分析了对错？
