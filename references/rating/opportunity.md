# 机会维度（Opportunity）

> 评估上涨空间与概率，星级越高机会越大

---

## 维度定义

| 属性 | 说明 |
|------|------|
| **含义** | 上涨空间与概率 |
| **星级说明** | 1⭐=很小，5⭐=极大 |
| **评估重点** | 趋势强度、催化剂、空间大小 |

---

## 评分规则

### 评分因素

| 因素 | 权重 | 说明 |
|------|------|------|
| **趋势强度** | 30% | 趋势行情 > 震荡行情 |
| **催化剂** | 30% | 明确催化剂 > 无催化剂 |
| **上涨空间** | 25% | 空间越大分数越高 |
| **概率** | 15% | 上涨概率评估 |

### 评分标准

| 星级 | 含义 | 典型特征 | 评分要点 |
|------|------|---------|---------|
| ⭐ | 机会很小 | 震荡市、方向不明 | 无明确趋势，催化剂缺失 |
| ⭐⭐ | 机会较小 | 反弹行情、空间有限 | 弱势反弹，空间<10% |
| ⭐⭐⭐ | 中等机会 | 趋势行情、正常空间 | 趋势明确，空间10-20% |
| ⭐⭐⭐⭐ | 机会较好 | 强势趋势、空间较大 | 强势趋势，空间20-40% |
| ⭐⭐⭐⭐⭐ | 绝佳机会 | 主升浪、催化剂明确 | 主升浪，空间>40%，催化剂强力 |

---

## 评分检查清单

### ⭐ 机会很小（1星）

- [ ] 震荡市，无明确方向
- [ ] 无催化剂或催化剂已兑现
- [ ] 空间<5%

### ⭐⭐ 机会较小（2星）

- [ ] 弱势反弹
- [ ] 有催化剂但不明确
- [ ] 空间5-10%

### ⭐⭐⭐ 中等机会（3星）

- [ ] 趋势明确
- [ ] 有催化剂待验证
- [ ] 空间10-20%

### ⭐⭐⭐⭐ 机会较好（4星）

- [ ] 强势趋势
- [ ] 催化剂明确
- [ ] 空间20-40%

### ⭐⭐⭐⭐⭐ 绝佳机会（5星）

- [ ] 主升浪中
- [ ] 催化剂强力且明确
- [ ] 空间>40%

---

## 评分流程

```
Step 1: 判断趋势
  - 主升浪？ → +2分
  - 强势趋势？ → +1.5分
  - 震荡市？ → +0.5分
  - 下跌趋势？ → +0分

Step 2: 评估催化剂
  - 政策+行业+公司共振？ → +2分
  - 行业+公司共振？ → +1.5分
  - 仅公司利好？ → +1分
  - 无催化剂？ → +0分

Step 3: 计算空间
  - 空间>40%？ → +2分
  - 空间20-40%？ → +1.5分
  - 空间10-20%？ → +1分
  - 空间<10%？ → +0分

Step 4: 综合评分
  合计：6分以上 → 5星
  合计：4-5分 → 4星
  合计：3分 → 3星
  合计：1-2分 → 2星
  合计：0分 → 1星
```

---

## 场景示例

### 案例1：主升浪龙头

```
趋势：主升浪 → 2分
催化剂：政策+行业共振 → 2分
空间：50%+ → 2分
合计：6分 → ⭐⭐⭐⭐⭐
```

### 案例2：超跌反弹

```
趋势：弱势反弹 → 0.5分
催化剂：技术超卖 → 0.5分
空间：15% → 1分
合计：2分 → ⭐⭐
```

### 案例3：震荡市

```
趋势：震荡 → 0.5分
催化剂：无 → 0分
空间：8% → 0分
合计：0.5分 → ⭐
```

## 机会维度 vs 其他维度关系

| 组合 | 评级影响 |
|------|---------|
| 机会高 + 确定高 | 强力买入信号 |
| 机会高 + 安全低 | 谨慎，虽有机会但风险大 |
| 机会低 + 舒适低 | 明确不买 |
