# 确定性维度（Certainty）

> 评估上涨逻辑清晰度，星级越高逻辑越清晰

---

## 维度定义

| 属性 | 说明 |
|------|------|
| **含义** | 上涨逻辑清晰度 |
| **星级说明** | 1⭐=模糊，5⭐=非常确定 |
| **评估重点** | 催化剂明确性、逻辑支撑、多维度共振 |

---

## 评分规则

### 评分因素

| 因素 | 权重 | 说明 |
|------|------|------|
| **催化剂明确性** | 35% | 催化剂是否清晰可验证 |
| **逻辑支撑** | 30% | 是否有多个维度支撑 |
| **历史验证** | 20% | 类似案例历史准确率 |
| **时间窗口** | 15% | 催化剂兑现时间是否明确 |

### 评分标准

| 星级 | 含义 | 典型特征 | 评分要点 |
|------|------|---------|---------|
| ⭐ | 很模糊 | 纯概念、无实质支撑 | 无催化剂，逻辑混乱 |
| ⭐⭐ | 较模糊 | 消息不明确、联动差 | 催化剂模糊，单一支撑 |
| ⭐⭐⭐ | 中等确定 | 有催化剂但需验证 | 催化剂存在但需观察 |
| ⭐⭐⭐⭐ | 较确定 | 多维度信号共振 | 多维度确认，逻辑清晰 |
| ⭐⭐⭐⭐⭐ | 非常确定 | 政策+业绩+技术共振 | 催化剂强力，多维度完美共振 |

---

## 评分检查清单

### ⭐ 很模糊（1星）

- [ ] 纯概念炒作
- [ ] 无实质催化剂
- [ ] 跟风炒作，逻辑不清

### ⭐⭐ 较模糊（2星）

- [ ] 有消息但不明确
- [ ] 单一维度支撑
- [ ] 板块联动性差

### ⭐⭐⭐ 中等确定（3星）

- [ ] 有明确催化剂
- [ ] 催化剂待验证
- [ ] 1-2个维度支撑

### ⭐⭐⭐⭐ 较确定（4星）

- [ ] 催化剂明确且可验证
- [ ] 多维度信号共振
- [ ] 板块联动良好

### ⭐⭐⭐⭐⭐ 非常确定（5星）

- [ ] 政策+行业+公司共振
- [ ] 业绩确定性增长
- [ ] 技术面完美配合
- [ ] 历史案例验证

---

## 多维度共振评估

| 共振维度数 | 确定性加分 |
|-----------|-----------|
| 政策 + 行业 + 公司 | +2分（满分） |
| 行业 + 公司 + 技术 | +1.5分 |
| 公司 + 技术 + 资金 | +1.5分 |
| 政策 + 行业 | +1分 |
| 仅公司利好 | +0.5分 |
| 无共振 | +0分 |

---

## 评分流程

```
Step 1: 催化剂评估
  - 政策+行业+公司共振 → +3分
  - 行业+公司共振 → +2分
  - 公司利好 → +1分
  - 无明确催化剂 → +0分

Step 2: 逻辑支撑评估
  - 多维度完美共振 → +2分
  - 2-3个维度支撑 → +1.5分
  - 单一维度 → +0.5分
  - 无支撑 → +0分

Step 3: 历史验证
  - 类似案例高胜率 → +1分
  - 无历史参考 → +0分

Step 4: 时间窗口
  - 催化剂即将兑现（<1周）→ +1分
  - 催化剂中期（1-4周）→ +0.5分
  - 催化剂长期（>4周）→ +0分

Step 5: 综合评分
  合计5分以上 → 5星
  合计4分 → 4星
  合计2-3分 → 3星
  合计1分 → 2星
  合计0分 → 1星
```

## 场景示例

### 案例1：政策利好龙头

```
催化剂：政策+行业+公司共振 → 3分
逻辑支撑：政策+技术+资金共振 → 2分
历史验证：同类政策利好高胜率 → 1分
时间窗口：政策发布当天 → 1分
合计：7分 → ⭐⭐⭐⭐⭐
```

### 案例2：纯概念炒作

```
催化剂：纯概念，无实质 → 0分
逻辑支撑：单一概念 → 0.5分
历史验证：无 → 0分
时间窗口：不确定 → 0分
合计：0.5分 → ⭐
```

### 案例3：业绩预增

```
催化剂：公司业绩预增 → 1分
逻辑支撑：业绩+技术共振 → 1.5分
历史验证：业绩预增准确率高 → 1分
时间窗口：年报发布前 → 0.5分
合计：4分 → ⭐⭐⭐⭐
```

## 确定性维度 vs 其他维度关系

| 组合 | 评级影响 |
|------|---------|
| 确定⭐ ≥ 3 + 安全⭐ ≥ 3 + 机会⭐ ≥ 3 | 买入信号 |
| 确定⭐ < 3 | 即使有机会也观望 |
| 确定⭐ ≥ 4 + 机会⭐ ≥ 4 | 强力买入 |
| 确定⭐ ≥ 4 + 机会⭐ < 3 | 机会不够，不买 |
